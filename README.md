# Lemon App
Lemon App 是一款开源（遵循GPL以及Anti 996协议）的基于QQ音乐的在线音乐播放器。
支持同步QQ音乐歌单及喜欢，更有排行版、分类歌单、歌手、电台等功能 发现好音乐！
